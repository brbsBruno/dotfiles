dotfiles
--

1. Clone this project
```
$ git clone https://github.com/brbsbruno/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
```

2. Setup!
```
$ sh ./setup
```

3. Add the following to your .zshrc:
```
source /usr/local/share/zsh-autosuggestions/zsh-autosuggestions.zsh
```
